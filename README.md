# Chat App

A real-time chat application built with React, Vite, Express, and Socket.IO. Users can enter a username, join a programming room, see online users, and send live messages.

## Features

- Real-time messaging with Socket.IO
- Join different chat rooms
- Online users list
- Username-based chat
- React frontend with Vite
- Express backend server

## Tech Stack

- React
- Vite
- React Router
- Socket.IO Client
- Node.js
- Express
- Socket.IO
